## 五、与未排除极值时的对比小结


| 维度        | 全量（37 条）      | 排除极值后（32 条）         | 说明                            |
| --------- | ------------- | ------------------- | ----------------------------- |
| 总展示量      | 3,074,333     | 155,013             | 被排除的 5 条为高展示量素材，剩余以中低展示为主     |
| 总花费       | $67,017       | $2,511              | 同上                            |
| 平均 CTR    | 13.52%        | 13.23%              | 接近，排除极值后略降                    |
| 平均 CVR    | 12.92%        | 12.63%              | 接近                            |
| 平均完播率     | 188.38%       | 192.99%             | 均存在 >100% 样本，需业务口径校验          |
| 标签「目标物品」  | 展示/花费约整体 7 倍+ | 展示约 2.3 倍、花费约 3.6 倍 | 极值中多为高消耗素材，排除后该标签相对优势减弱但仍高于整体 |
| 标签「点消/拖消」 | 展示远低于整体       | 排除极值后展示比 1.2～2.6    | 在「去极值」后的整体中，各标签展示量比更集中        |


- 排除极值后，**整体均值**（如平均 CTR、CVR）更贴近「大多数素材」的水平，不被少数极高展示素材拉偏。
- **标签对比**的比值是在「排除极值后的整体」基础上计算；原先「目标物品」的 7 倍优势部分来自被剔除的高展示素材，剔除后三者差异收窄但仍可区分。
- **相关性**在去掉极值后，线性/秩关系更反映主体分布，可对比 `tag_correlation_*.csv` 与 `tag_no_outliers_correlation_*.csv` 查看差异。
